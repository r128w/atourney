# Aura Tourney
WIP competitive bullet hell multiplayer game etc


## How to run
Download the whole repo, ```cd``` to the directory, and run
```bash
python3 server.py
```
The server hosts content on port 8000, and prints out the IP with which it can be accessed by other devices on startup. ```localhost:8000``` or ```127.0.0.1:8000``` also work.
Players join by going to the site.
